# 🛢️ PID AutoTuner - Система Автоматической Настройки ПИД-Регулятора

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![C++17](https://img.shields.io/badge/C%2B%2B-17-blue.svg)](https://isocpp.org/)
[![Qt6](https://img.shields.io/badge/Qt-6.x-green.svg)](https://www.qt.io/)

Профессиональное приложение для автоматической настройки ПИД-регуляторов в нефтегазовой отрасли.

## ✨ Особенности

- **Множественные методы настройки**: Ziegler-Nichols, Cohen-Coon, IMC
- **Адаптивный Anti-Windup**: Патентоспособный алгоритм защиты от насыщения
- **Модели процессов**: FOPDT, теплообменники, трубопроводы, резервуары
- **Визуализация**: Интерактивные графики на Qt Charts
- **Экспорт данных**: CSV, JSON форматы

## 🚀 Быстрый Старт

### Требования
- Qt 6.x (Widgets, Charts)
- CMake 3.16+
- C++17 компилятор

### Сборка
```bash
mkdir build && cd build
cmake ..
make -j$(nproc)
./PIDAutoTunerOilGas
```

## 📖 Использование

1. Введите параметры процесса (K, T, L)
2. Выберите метод настройки
3. Нажмите "Рассчитать Настройку"
4. Запустите симуляцию
5. Экспортируйте результаты

## 🔬 Патентоспособные Инновации

1. **Адаптивный Anti-Windup Алгоритм**
2. **Интеллектуальный Выбор Метода**
3. **Bumpless Переключение Режимов**
