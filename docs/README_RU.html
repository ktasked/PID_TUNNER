🛢️ PID AutoTuner для Нефтегазовой Отрасли

📋 Описание Проекта

Профессиональное приложение для автоматической настройки ПИД-регуляторов, специально разработанное для нефтегазового сектора.

✨ Ключевые Особенности

  • Множественные методы настройки: Ziegler-Nichols, Cohen-Coon, IMC
  • Адаптивный Anti-Windup: Патентоспособный алгоритм защиты от насыщения
  • Модели процессов: FOPDT, теплообменники, трубопроводы, резервуары
  • Визуализация: Интерактивные графики на Qt Charts
  • Экспорт данных: CSV, JSON форматы

🚀 Быстрый Старт

Требования

  • Qt 6.x (Widgets, Charts)
  • CMake 3.16+
  • C++17 совместимый компилятор

Сборка


mkdir build && cd build
cmake ..
make -j$(nproc)
./PIDAutoTunerOilGas

📖 Использование

  1. Ввод параметров процесса:
    • Коэффициент усиления (K)
    • Постоянная времени (T)
    • Время задержки (L)
  2. Выбор метода настройки:
    • Ziegler-Nichols - классический, быстрый отклик
    • Cohen-Coon - для процессов с большой задержкой
    • IMC - оптимальный баланс робастности
  3. Расчёт параметров: Нажмите "Рассчитать Настройку"
  4. Симуляция: Запустите симуляцию для проверки качества
  5. Экспорт: Сохраните результаты в CSV/JSON

🔬 Патентоспособные Инновации

1. Адаптивный Anti-Windup Алгоритм

Динамическая корректировка коэффициента отслеживания на основе:

  • Темпа изменения ошибки
  • Исторического тренда
  • Текущего состояния насыщения

2. Интеллектуальный Выбор Метода Настройки

Автоматическая рекомендация оптимального метода на основе:

  • Отношения L/T (задержка/постоянная времени)
  • Уровня шума измерений
  • Типа технологического процесса

3. Bumpless Переключение Режимов

Безударный переход между ручным и автоматическим режимами с компенсацией интегральной составляющей.

📊 Поддерживаемые Процессы

  • FOPDT: Первый порядок + задержка (наиболее распространен в нефтегазе)
  • Теплообменники: Специфичная модель теплопередачи
  • Трубопроводы: Гидравлическая модель с учетом вязкости
  • Резервуары: Интегрирующие звенья для уровня жидкости
